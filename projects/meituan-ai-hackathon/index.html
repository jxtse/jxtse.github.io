技术架构

FastAPI + 单会话 Agent + SSE 实时事件流 + Mock 美团/点评数据层。

🧠 Agent 与交互

  • 单会话状态机，分段推进：/api/start → accept / reject / switch → execute
  • SSE /api/events 实时推 thinking / stage / tool / self_heal 事件到前端思考链路
  • LLM 接口 OpenAI 兼容，可对接任意兼容网关

🛡️ 三类异常自愈

  • no_table 无座 → 同人群/饮食约束切换可订餐厅
  • sold_out 无票 → 同类顺路活动替换，保留时间轴
  • conflict 冲突/超窗 → 触发规划器整体重排

⏱️ 硬性指标（赛题约束）

  • 方案生成 < 30s · 单工具 < 3s · 端到端 < 2min
  • 至少处理 3 类异常 · Pass@1
  • 商户数据全 Mock，不接真实美团
